# 🔐 Sistema de Login com Flask

Projeto de login profissional desenvolvido com Flask, HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Cadastro de usuário
- Login
- Validação de senha forte
- Barra de força da senha
- Modo escuro com animação
- Banco de dados SQLite

## 🛠 Tecnologias usadas

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## ▶️ Como executar

1. Instale as dependências:
   pip install flask

2. Rode o projeto:
   python app.py

3. Acesse:
   http://127.0.0.1:5000
